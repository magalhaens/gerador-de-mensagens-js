# Gerador de Mensagens

Projeto simples desenvolvido com HTML, CSS e JavaScript para gerar mensagens personalizadas a partir de campos preenchidos pelo usuário.

## Sobre o projeto

O Gerador de Mensagens permite que o usuário informe:

- Nome da pessoa
- Assunto
- Mensagem principal

Após preencher os campos, a aplicação gera uma mensagem organizada na tela.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Captura de dados digitados pelo usuário
- Validação para impedir campos vazios
- Geração dinâmica de mensagens
- Limpeza automática dos campos após gerar a mensagem
- Foco automático no campo de nome
- Uso de `textContent` para inserir texto de forma mais segura na página
- Layout responsivo
